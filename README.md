# Hollywood Bets AI Horse Racing Predictor

An advanced AI-powered system for predicting South African horse racing winners using data from Hollywoodbets.

## Features

- **Web Scraping**: Automated scraping of race cards, results, and tipster data from Hollywoodbets
- **Machine Learning**: Advanced ensemble models (XGBoost, LightGBM, CatBoost, Random Forest) for predictions
- **Continuous Learning**: Self-improving AI that updates with new race results
- **Tipster Analysis**: Incorporates insights from the most accurate tipsters
- **Punters Challenge**: Automated strategy generation for Punters Challenge
- **Pick Six**: Smart combination generation for Pick Six bets
- **Database Storage**: SQLite/PostgreSQL database for historical data
- **Real-time Predictions**: Daily predictions for upcoming races
