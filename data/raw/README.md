# Raw Data Directory

This directory contains raw scraped data from Hollywoodbets.

## Files Structure:
- `race_cards/` - Raw race card data
- `results/` - Raw race results
- `tipsters/` - Raw tipster data
- `form_guides/` - Raw form guide data

## Data Formats:
- JSON files for structured data
- HTML files for raw scraped pages
- CSV files for tabular data
