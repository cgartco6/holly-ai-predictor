# Models Directory

This directory contains trained machine learning models.

## Files Structure:
- `ensemble_model.joblib` - Main ensemble model
- `xgboost_model.joblib` - XGBoost model
- `lightgbm_model.joblib` - LightGBM model
- `scaler.joblib` - Feature scaler
- `encoder.joblib` - Label encoder
- `feature_selector.joblib` - Feature selector
